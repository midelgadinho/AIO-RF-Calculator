A simple (almost) All-In-One RF calculator, suitable for telecom engineers and students.

It has various calculators designed to help in Radio-Frequency projects.

At the moment, this are the available calculators:

- Frequency / Wavelength Calculator
- Line of Sight Calculator
- Units Converter (Metric <-> Imperial)
- Watts <-> dBm
- Isotropic Converter (dBd <-> dBi)
- ERP / EIRP Calculator
- Field Strength Calculator
- Cable Loss Calculator
- Free Space Path Loss and Received Power Calculator
- SNR Calculator

If you have any issues while using this software, feel free to open an issue.
